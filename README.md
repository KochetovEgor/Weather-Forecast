Этот проект позволяет с помощью простейшего веб-интерфейса выбрать точку на Земле, в которой мы хотим узнать погоду за определённый промежуток времени, который определяется по начальной и конечной датам. 
Результатом работы будет список, состоящий из дней и минимальной и максимальной температур за эти дни в выбранной пользователем точке Земли.
